## To run

### Server side (Unix)

```bash
cd server
pip3 install -r requirements.txt
python3 main.py 
```

### Client Side (Raspberry Pi 3B used)

***Note*** - **Change the ip address before running**

```bash
cd client
python3 main.py
```
